## 1. 产品概述

青藏高原草甸科考点冻土活动层厚度与地温梯度剖面对比看板，用于科研人员可视化分析不同坡向、不同海拔的冻土监测剖面数据对比。解决青藏高原冻土研究的数据采集、分析、对比的全流程数字化管理。

- **主要用途**：冻土活动层厚度与地温梯度的多维度对比分析
- **目标用户**：青藏高原草甸科考点科研人员、生态研究人员
- **核心价值**：将原始采样数据转化为直观的可视化对比，支持科研决策

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 科研人员 | 无需注册，本地部署 | 数据查询、筛选、可视化分析、数据导入导出 |

### 2.2 功能模块

1. **数据看板主页**：坡向筛选、日期范围选择、多图表联动展示
2. **剖面数据对比**：活动层厚度柱状图、地温差趋势、邻剖面差值散点图
3. **剖面布设示意**：地理空间位置示意图
4. **数据管理**：采样数据导入、筛选结果导出

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|---------|---------|---------|
| 看板主页 | 筛选区域 | 坡向多选筛选、日期范围选择器、数据刷新按钮 |
| 看板主页 | 统计卡片 | 展示当前筛选条件下的关键指标（平均厚度、平均温差、剖面数量） |
| 看板主页 | 厚度柱状图 | 各剖面活动层厚度日均值对比柱状图 |
| 看板主页 | 地温差折线图 | 各地温梯度(50cm-20cm)日均值对比 |
| 看板主页 | 邻剖面差值散点图 | 同坡向相邻剖面活动层厚度均值差值散点图 |
| 看板主页 | 剖面布设示意图 | 按坡向分组的剖面空间位置示意 |
| 数据导入 | 采样导入 | CSV/Excel格式新采样数据批量导入 |
| 数据导出 | 结果导出 | 当前筛选条件下的分析结果CSV导出 |

## 3. 核心流程

用户进入看板后，首先选择坡向筛选条件，设置日期范围，系统自动计算各剖面的活动层厚度日均值与地温差日均值，同时计算同坡向相邻剖面的厚度差值。用户可通过多维度图表进行对比分析，支持导入新采样数据或导出当前分析结果。

```mermaid

    A["进入看板"] --> B["选择坡向筛选"]
    B --> C["设置日期范围"]
    C --> D["系统计算日均值"]
    D --> E["计算邻剖面差值"]
    E --> F["多图表联动展示"]
    F --> G["数据导入/导出"]
```

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深青色(#0F766E)代表高原冻土，暖橙色(#F97316)代表地温数据对比
- **辅助色**：冷灰色系作为背景，冰川蓝(#38BDF8)作为强调色
- **按钮风格**：圆角中等，悬停有微动画
- **字体**：无衬线字体，标题粗体，数据等宽字体
- **布局风格**：卡片式网格布局，顶部筛选栏，下方四象限图表区
- **风格定位**：科研专业风格，简洁严谨，数据驱动

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|---------|---------|--------|
| 看板主页 | 筛选区域 | 顶部固定筛选栏，下拉多选坡向标签，日期范围选择器，操作按钮组 |
| 看板主页 | 统计卡片 | 渐变背景卡片，大号数字指标，趋势箭头指示 |
| 看板主页 | 图表区域 | 2x2网格布局，每个图表带标题、说明、导出按钮 |
| 看板主页 | 剖面示意图 | 左侧坡向分组，海拔高度Y轴，剖面点标注 |

### 4.3 响应式设计

- 桌面端优先，采用12列栅格布局
- 平板端自适应为2列布局
- 移动端单列堆叠，图表可横向滚动
- 图表支持触摸缩放

### 4.4 交互细节

- 图表hover显示详细数据tooltip
- 筛选条件变更时图表平滑过渡动画
- 点击剖面点联动高亮对应数据
- 导入导出按钮悬浮提示
